ETAPA 01 · COMPLETADO ✓

Crear las tablas base para un modelo relacionable

Objetivo: estructurar los datos antes de tocar Power Pivot

Genera 4 tablas de ejemplo para un análisis de ventas: "Ventas" (con ID_Producto, ID_Cliente, Fecha, Cantidad, Importe), "Productos" (ID_Producto, Nombre, Categoría, Costo), "Clientes" (ID_Cliente, Nombre, Región, Segmento) y "Calendario" (Fecha, Año, Mes, Trimestre). Asegúrate de que los campos clave (ID_Producto, ID_Cliente, Fecha) tengan el mismo nombre y tipo de dato en todas las tablas para que las relaciones se puedan crear sin conflictos. Colócalas en hojas separadas y dales formato de Tabla de Excel. Importante: la columna Fecha debe cargarse como un valor de fecha real de Excel (formato de fecha, no texto) tanto en "Calendario" como en "Ventas". Evitá guardarla como texto en formato ISO (AAAA-MM-DD), ya que eso impide relacionarla correctamente en Power Pivot.

Copiá el prompt y pegalo en tu asistente de IA dentro de Excel.
ETAPA 02 · COMPLETADO ✓

Incorporar las tablas al Modelo de Datos de Power Pivot

Objetivo: cargar y validar los datos antes de crear relaciones

Toma estas 4 tablas de Excel y agrégalas al Modelo de Datos de Power Pivot. Antes de crear relaciones, revisa que los tipos de datos de los campos clave (ID_Producto, ID_Cliente, Fecha) coincidan entre tablas, y avísame si detectas alguna inconsistencia.

Aclaración: la IA no puede agregar tablas al Modelo de Datos de Power Pivot directamente — esa acción no está disponible desde su interfaz. Lo que sí hace es validar los tipos de datos de las claves y sugerir las relaciones. El paso de agregar las tablas al modelo se hace manualmente en Excel: seleccioná cada tabla y usá Power Pivot → Agregar al Modelo de datos.
Copiá el prompt y pegalo en tu asistente de IA dentro de Excel.
ETAPA 03

Crear las relaciones entre las tablas

Objetivo: conectar el modelo con relaciones 1 a muchos

Ahora crea las relaciones entre las tablas en Power Pivot, explicando por qué cada relación es 1 a muchos y cuál es la tabla "del lado uno" en cada caso. Si no puedes crear las relaciones directamente desde tu interfaz, no modifiques nada del libro: solo indícalo y sugiere las relaciones que corresponden.

Prompt principal: la IA valida y sugiere las relaciones.
Aclaración: hasta el momento, la IA no puede crear las relaciones directamente en Power Pivot — esa acción no está disponible desde su interfaz. Las relaciones hay que crearlas "a mano" en la vista de diagrama de Power Pivot. Para eso, usá el siguiente prompt, que le pide a la IA el paso a paso exacto para hacerlo vos mismo.

Indícame paso a paso cómo crear en Power Pivot las relaciones entre tblVentas, tblProductos, tblClientes y tblCalendario, especificando qué campo conecta cada par de tablas, por qué cada relación es 1 a muchos, y cuál es la tabla "del lado uno" en cada caso.

Prompt de apoyo: la IA te guía para crear las relaciones manualmente.
